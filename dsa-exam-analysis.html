MUST PREPARE 7 Marks 2025

Define ADT (Abstract Data Type). Explain why it is needed and list different types of data structures with examples.

Model Answer

Definition: An Abstract Data Type (ADT) is a mathematical model for data types where the data type is defined by its behavior (semantics) from the point of view of a user of the data.

Explanation: ADT provides a clear separation between the logical properties of data and its physical implementation.

Linear Data Structures: Array, Stack, Queue, Linked List
Non-Linear Data Structures: Tree, Graph

Exam Tip

Always mention that ADT defines 'what' to do, while Data Structure defines 'how' to do it.
